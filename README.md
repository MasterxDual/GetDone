# Proyecto Desarrollo Web - GetDone

## Estructura General del Proyecto
```
task-manager/
├── frontend/               # Parte del cliente
│   ├── assets/             # Archivos estáticos (imágenes, fuentes)
│   ├── css/                # Estilos CSS
│   ├── js/                 # JavaScript del frontend
│   ├── views/              # Vistas HTML
│   │   ├── auth/           # Vistas de autenticación
│   │   ├── admin/          # Vistas para administrador
│   │   ├── user/           # Vistas para usuario normal
│   │   └── shared/         # Componentes compartidos
│   └── index.html          # Página principal
├── backend/                # Parte del servidor
│   ├── config/             # Configuraciones
│   ├── controllers/        # Controladores
│   ├── models/             # Modelos de datos
│   ├── routes/             # Rutas
│   ├── middlewares/        # Middlewares
│   └── app.js              # Aplicación principal
├── database/               # Scripts y modelos de base de datos
├── docs/                   # Documentación
└── README.md               # Documentación principal
```
